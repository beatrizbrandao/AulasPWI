Tags HTML: elementos para criar sua página

Elementos basicos
Tag Funcionalidade
HTML Ele é a estrutura, será usado para o inicio de todas as tags e irá conte-las,ou seja,todos os outros elementos irá ser descendente dele
HEAD Irá conter todas as informações, que serão importantes, principlamente, para seu navegador, como por exemplo o título de uma página através de uma tag colocada.
TITLE Essa tag será útil para criar um titúlo para sua página, como dito acima
BODY É o corpo do seu documento, onde todas as informações colocadas dentro dela será vísivel no site
<-- --> Tudo que for escrito dentro dessa tag não será visto no site
