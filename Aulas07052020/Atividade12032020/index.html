Tags HTML: elementos para criar sua pagina

Elementos basicos

Tag Funcionalidade
HTML Ele e a estrutura, sera usado para o inicio de todas as tags e ira conte-las,ou seja,todos os outros elementos ira ser descendente dele
HEAD Ira conter todas as informacoes, que serao importantes, principlamente, para seu navegador, como por exemplo o titulo de uma pagina atraves de uma tag colocada.
TITLE Essa tag sera util para criar um titulo para sua pagina, como dito acima
BODY E o corpo do seu documento, onde todas as informacoes colocadas dentro dela sera visivel no site
<-- --> Tudo que for escrito dentro dessa tag nao sera visto no site
h1 Ele vai indicar um titulo em um site
main Define o conteudo principal dentro do body
